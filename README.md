## Build and run

This project is configured as a Gradle application using JDK 25 and OpenJFX 25.0.3. The application entry point is set by the `mainClass` property in `gradle.properties`.

To build and test the project:

```bash
./gradlew build
```

To run the JavaFX application after replacing the placeholder `mainClass` value:

```bash
./gradlew run
```
